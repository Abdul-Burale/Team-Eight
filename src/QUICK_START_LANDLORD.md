## ğŸ†˜ Troubleshooting

### "Please sign in to create a listing"
**Solution**: You're not signed in. Click "Sign In" and use landlord credentials.

### "Only landlords can create property listings"
**Solution**: Your account is not set as "Landlord" type. Contact support or create new landlord account.

### "Failed to create property"
**Solution**: 
- Check all required fields are filled
- Verify price is a valid number
- Ensure bedrooms/bathrooms are numbers
- Check internet connection

### "Failed to fetch dashboard data"
**Solution**:
- Click "Refresh" button
- Check internet connection
- Sign out and sign back in
- Clear browser cache

### Property not appearing in list
**Solution**:
- Wait a few seconds and refresh
- Check "My Listings" tab
- Verify property was created successfully
- Check browser console for errors

### QR Code not generating
**Solution**:
- Ensure property has an ID
- Refresh the page
- Try closing and reopening QR dialog
- Check browser supports canvas
