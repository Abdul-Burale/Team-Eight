## ğŸ¨ Color & Style Guide

### Primary Colors
- **Blue (#2563eb)**: Primary actions, links, default badges
- **Green (#16a34a)**: Success, accepted, paid, completed
- **Yellow (#eab308)**: Warning, pending, under review
- **Red (#dc2626)**: Danger, rejected, overdue, delete
- **Purple (#9333ea)**: Premium features, revenue, investment
- **Gray (#6b7280)**: Neutral, disabled, secondary text

### Typography
- **Headings**: Large, bold, clear hierarchy
- **Body text**: 14-16px, readable, good line height
- **Labels**: 12-14px, gray color, uppercase for emphasis
- **Numbers/Stats**: Large (24-32px), bold, colored

### Spacing
- **Cards**: 16-24px padding
- **Grid gaps**: 16-24px between items
- **Sections**: 32-48px vertical spacing
- **Mobile**: Reduce padding by 25-50%

### Components
- **Cards**: White background, subtle shadow, rounded corners
- **Buttons**: Rounded, adequate padding, clear labels
- **Badges**: Small, colored, rounded pills
- **Tables**: Alternating rows, clear headers, responsive
- **Forms**: Clear labels, validation, helpful hints

## âœ¨ Interactive Elements

### Buttons
- **Primary**: Blue background, white text, hover effect
- **Secondary**: White background, border, gray text
- **Success**: Green background for positive actions
- **Danger**: Red background for destructive actions
- **Icon buttons**: Icon + text for clarity

### Badges
- **Status badges**: Color-coded (green, yellow, red)
- **Count badges**: Small, circular, on icons
- **Info badges**: Outline style for neutral info

### Dialogs/Modals
- **Create listing**: Multi-step form
- **Make offer**: Form with calculations
- **Application**: Document upload + form
- **Messages**: Compose + inbox view

### Tables
- **Desktop**: Full table with all columns
- **Tablet**: Hide less important columns
- **Mobile**: Convert to cards or scrollable
